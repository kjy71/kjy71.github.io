- 긴문장의 경우 다음단어의 첫문자는 대문자
  ex) myName camelCase라고함  //  by 파이썬 my_name_is snake_case라고함(const도 생략됨)

- const : 값 update 불가
  let : 값 update 가능

- array - 하나의 variable안에 데이터의 list를 가지는 것

    * push : 배열명.push("sun") - 배열 끝에 sun 추가

- function name() : ()는 실행의 의미
               function의 결과값은 바로 variable에 담지못하고 return으로 
               return은 function의 종료

- object : const object = { name : a , 
                            age : 20,
                            hobby : function(){}
                             ...};
           const object안의 요소는 update 가능

- typeof : variable의 type을 알려줌

- parseInt() : string을 숫자로 변환

- NaN : Not a Number / isNaN : 숫자가 아니면 true, 숫자이면 false

- console.dir : element 확인가능
                element 안에 on으로 시작하는 property는 event listener 임 
                사용할때는 on을 빼고 사용
                click, mouseenter, mouseleave

- document.getElementById("id");
  document.getElementByClassName("classname");
   document.getElementByTagName("h1,div,button,anchor...");

- document.querySelector : element를 css방식으로 검색
   
   선택자 종류
    * class : .
    * id : #
    * div :div.
    * 모든태그 : h2 등등
    * 요소선택 : input[type="text"]

- addEventListener("click",function)
   * function뒤에 ()를 넣지 않음으로써 실행을 listener에게 넘김
   * title.addEventListener("click",function); = title.onclick = function;
   * function에 기본적으로 첫번째 argument로 event정보를 넘겨줌

- classList : class의 옵션개념 (클래스 이름은 여러개가 가능해)
    * .contains(classname) : classname이 있어?
    * .remove(classname) : classname을 지워
    * .add(classname) : classname 추가해
    * .toggle(classname) : 위 3가지를 한번에 - 있으면 지우고 없으면 생성
